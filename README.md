# Agentic AI

Agentic AI is a full-stack chatbot project that lets you build and interact with task-oriented AI agents through a Streamlit interface backed by a FastAPI service. The application supports Groq and OpenAI chat models, custom system prompts, and optional web search via Tavily, making it useful for experimenting with retrieval-enabled assistants and agent workflows.

## What It Does

- Creates configurable AI agents with custom system prompts
- Supports multiple LLM providers, including Groq and OpenAI
- Exposes a FastAPI chat endpoint for backend orchestration
- Provides a Streamlit UI for interactive testing
- Optionally enables web search for more grounded responses

## Tech Stack

- Python 3.11+
- FastAPI
- Streamlit
- LangGraph
- LangChain
- Groq and OpenAI model integrations
- Tavily search tool
